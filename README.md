# Bank Admin CLI üè¶

A simple bank management system built with Python and SQLite.  
Features a **colorful command-line interface** using Colorama.

## Features
- Open new account
- View account information
- Withdraw and deposit money
- Transfer money between accounts
- Check account balance
- Command-line interface with colors

## Tech Stack
- Python
- SQLite
- Colorama

## Installation & Run
```bash
# Clone the repository
git clone https://github.com/Tahahalimi/bank-admin-cli.git
cd bank-admin-cli

# Install Colorama for colored terminal output
pip install colorama

# Run the program
python main.py
